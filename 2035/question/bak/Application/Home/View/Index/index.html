活动介绍

上海市第六次规划土地工作会议于不久前召开，标志着上海新一轮城市总体规划编制工作正式启动。新一轮城市总体规划的期限确定为2040年，将指导上海未来20多年的城市发展、建设和管理。“开门做规划”是本次规划的最大亮点，社会公众、专家和媒体将全程参与。

“上海，我的2040”愿景调查旨在获悉您对上海2040年发展的期许。您的意见对上海新一轮城市总体规划编制非常重要。愿广大市民积极参与，为建设我们的美好家园出谋划策。

衷心感谢您的支持，谢谢！

调查说明

1、 本次愿景调查将分三期进行。目前正在进行第一期问卷调查，活动时间截止至8月2日。8月上旬我们将陆续推出第二期和第三期问卷调查。

2、 本次愿景调查的结果仅用于统计分析，您的资料不会被泄漏给其他任何部门和个人。

3、 第一期问卷为13道选择题，分为单选和多选题。单选题只能选择1个选项，多选题可以选择1个及1个以上选项。

参与有奖

1、 调查问卷结束后，请留下您的联系方式。每位成功填写问卷和联系方式的市民将有机会参与抽奖。

2、 每期活动我们将随机抽取2位中奖者，中奖者将获得300元手机话费。

3、 每期活动结束后我们将公布中奖名单（手机前三位和后四位），并在名单公布后的15个工作日内为中奖者完成手机充值。
